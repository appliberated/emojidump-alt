emojidump Help

Type options in the text input field at the top of the app, and press or tap Enter to execute the command. You can also click or tap the emojidump label in the top left corner of the app to (re)execute the command.

Example command (click or tap to use):

emojidump -u:6.0 -s -m:1000 -z:1

This command will dump 1,000 (random) Unicode 6.0 emojis, in random order, with the default space separator, and a 1x zoom.

Usage: emojidump [OPTION]...

-u:VERSION
Filter emojis by Unicode version. Valid values are 1.1, 3.0, 3.2, 4.0, 4.1, 5.1, 5.2, 6.0, 6.1, 7.0, 8.0, 9.0, 10.0, or 11.0.
-s
Display emojis in random order (shuffle).
-m:MAX
Specify a maximum number of emojis to display.
-j
Join emojis instead of separating them with spaces (which is the default behaviour).
-z:LEVEL
Sets the "zoom" level. The specified value must be between 1 and 7.
